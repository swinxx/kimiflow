## Function-by-function contract

### `_base_url_from_provider(provider)` (Bash 871-876)
`path = provider.vault_path // ""`; `url = path if path != "" else (provider.detection.url // "")`; if `url` empty -> `"https://127.0.0.1:27124"`. Return url.

### `provider_setup_plan_json(provider, setup_host)` (Bash 890-993)
`setup_host` not in `{codex, claude, all}` -> `"all"`. `raw_url = _base_url_from_provider(provider)`. `base = _normalize_loopback_origin(raw_url)`: if not None -> `base_url=base`, `mcp_url=base+"/mcp/"`, `status="setup_plan"`, `reason=""`; else `base_url=raw_url`, `mcp_url=""`, `status="blocked_non_loopback"`, `reason="non_loopback_url"`. `helper_path="~/.kimiflow/obsidian-mcp-headers.sh"`. `codex_snippet` = the 4-line printf (`[mcp_servers.obsidian]\nurl = "<mcp_url>"\nbearer_token_env_var = "OBSIDIAN_API_KEY"\ndefault_tools_approval_mode = "prompt"\n`). `claude_snippet` = `{mcpServers:{obsidian:{type:"http", url: mcp_url, headersHelper: helper_path}}}`. `manual_steps` = the fixed 6-item list. Emit the big object EXACTLY (Bash 936-992): key order + every conditional (`status == "blocked_non_loopback"` gates `mcp.url`, `mcp.certificate_url` (also requires `base_url` startswith `https://`), `helpers.*`, `hosts.codex.snippet`, `hosts.claude.snippet` (`{}` when blocked, else the object), `next_command`). `reason: null` when `reason==""`. `blocked: status=="blocked_non_loopback"`. provider_state reads `.configured/.available/.health.status/.auth.status/.detection.url/.vault_path` with `// default`. The helper-command strings interpolate `base_url` and `(if host=="all" then "all" else host end)`.

### `write_provider_prefetch_markdown(path, obj)` (Bash 4115-4129)
`makedirs(dirname)`; build: `# Vault Provider Prefetch\n\n`, `Generated: <iso_now()>\n\n`, then `Provider: <obj.provider.type>\n`, `Available: <obj.provider.available>\n` (bool->true/false), `Health: <obj.provider.health.status // "unknown">\n`, `Auth: <obj.provider.auth.status // "unknown">\n`, `Direct search ready: <obj.direct_search_ready == true>\n` (bool), `Query: <obj.query>\n\n`, then the fixed policy paragraph + `\n`. Write via `store.atomic_write`.

### `write_provider_sync_markdown(path, obj)` (Bash 4131-4158)
Like prefetch: header + `Generated` + Provider/Available/Health/Auth, `Direct write ready: <obj.direct_write_ready == true>\n`, `Candidates exported: <obj.candidates.exported_count // obj.candidates.count>\n`, `Total pending: <obj.candidates.count>\n`, `Omitted: <obj.candidates.omitted_count // 0>\n\n`, the fixed policy paragraph + `\n\n`. Then if `(candidates.exported_count // count) == 0` -> `No new publish-safe learning candidates are pending for Vault sync.\n`; else `## Candidates\n\n` + per `candidates.rows[]`: `"- [" + (topic//"uncategorized") + " · " + (kind//"learning") + " · " + (id//"") + "] " + ((summary//"") stringified, newlines->spaces, [:260]) + " (evidence: " + ((evidence//[])[0] // "NOT VERIFIED") + ")"` each + `\n`. Write via `store.atomic_write`.

### `run(argv)` (Bash 4160-4383)
`action = argv[0] if argv else "status"`; if argv, drop it. Parse `--root`/`--type`(default obsidian)/`--available`/`--path`/`--query`/`--host|--target`(default all)/`--write`/`--pretty`/`--help`/unknown->`return die("provider: unknown argument: <a>", 2)`. `root = resolve_root(root)`. `project = root/.kimiflow/project`; `manifest = project/VAULT-PROVIDER.json`; `now = iso_now()`. Dispatch on action (exact per Bash 4186-4381):
- **status:** `out = status_json(manifest)`.
- **health:** `provider = status_json(manifest)`; `out = {schema_version:1, status:(provider.health.status//"unknown"), recommended_action:(provider.health.recommended_action//"open_obsidian"), health:provider.health, auth:provider.auth, detection:provider.detection, capabilities:provider.capabilities, provider}`.
- **setup:** `provider = status_json(manifest)`; `out = provider_setup_plan_json(provider, setup_host)`.
- **detect/connect:** `detection = detection_json()`; if not `detection.available == true` -> `out = {schema_version:1, status:"not_detected", written:false, path:".kimiflow/project/VAULT-PROVIDER.json", detection, provider:null}`. Else: if action=="connect" -> write=1; if write: build the connect manifest object (keys 4234-4243: schema_version, type:"obsidian", available:true, mode:(existing.mode//"local-first"), vault_path:(detection.url), last_prefetch_at:(existing.last_prefetch_at//null), last_write_at:(existing.last_write_at//null), synced_learning_ids:(existing array-or-[]), detection, updated_at:now) where `existing = manifest_json(manifest)`, write it (atomic pretty). Then `provider = status_json(manifest)`; `out = {schema_version:1, status:(if write then "connected" else "detected"), written:(write==1), path, detection, provider}`.
- **configure:** map `--available` (`1|true|TRUE|yes|YES`->true; `0|false|FALSE|no|NO|""`->false; else `return die(..., 2)`); build the manifest object (keys 4276-4284: schema_version, type, available, mode:"local-first", vault_path, last_prefetch_at:null, last_write_at:null, synced_learning_ids:[], updated_at:now), write it; `out = status_json(manifest)`.
- **prefetch:** `provider = status_json(manifest)`; `prefetch_path = project/VAULT-PREFETCH.md`. If not `provider.available == true` -> `out = {schema_version:1, status:"skipped", reason:"provider_unavailable", path:".kimiflow/project/VAULT-PREFETCH.md", provider}`. Else `query = query or "project memory recall"`; `out = {schema_version:1, status:"prefetch_handoff", query, path, provider, direct_search_ready:(provider.health.direct_search_ready==true), review_required:true}`; if write: `write_provider_prefetch_markdown(prefetch_path, out)`; update manifest (`last_prefetch_at=now, updated_at=now, available=true`); `out["written"]=true`.
- **sync:** `provider = status_json(manifest)`; `sync_path = project/VAULT-SYNC.md`. If not available -> `out = {schema_version:1, status:"skipped", reason:"provider_unavailable", path, provider, candidates:{count:0, exported_count:0, omitted_count:0, ids:[]}}`. Else `sync_max = KIMIFLOW_PROVIDER_SYNC_MAX` (non-digit/empty->20; `<=0`->20); `candidates = _sync_candidates(root, project/LEARNINGS.jsonl, manifest)`; `export_candidates = candidates[:sync_max]`; `omitted = len(candidates) - len(export)`; `out = {schema_version:1, status:"sync_handoff", path, provider, direct_write_ready:(provider.health.direct_write_ready==true), review_required:true, candidates:{count:len(candidates), exported_count:len(export), omitted_count:omitted, ids:[c.id for c in export]}, written:false}`; if write: `handoff = {**out, candidates:{**out.candidates, rows:export}}`; `write_provider_sync_markdown(sync_path, handoff)`; update manifest (`last_write_at=now, updated_at=now, available=true, synced_learning_ids=unique(existing+export ids)`); `provider = status_json(manifest)`; `out["written"]=true; out["provider"]=provider`.
- **else:** `return die("provider action must be status, health, setup, detect, connect, configure, prefetch, or sync", 2)`.
`json_print(out, pretty)`; return 0.

**Manifest in-place update fidelity:** the prefetch/sync updates read `manifest_json(manifest)` (default-shaped object), mutate ONLY existing keys (order preserved) + recompute `synced_learning_ids = sorted(set(existing_list + new_ids))` (`unique`), then `atomic_write(pretty + "\n")`. The connect/configure paths build a fresh object with the exact Bash key order.
